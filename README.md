# markowitz-optimization-monte-carlo
Progetto di Finanza Quantitativa. Implementa l'Ottimizzazione di portafoglio di Markowitz su dati storici reali (yfinance) tramite l'algoritmo SLSQP di SciPy, calcolando la Frontiera Efficiente. Include una Simulazione Monte Carlo a 1000 scenari basata sul Moto Browniano Geometrico per l'analisi del rischio.
